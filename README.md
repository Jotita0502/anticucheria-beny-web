# Anticuchería Beny - Sistema Web de Pedidos

## Descripción

Sistema web desarrollado para la gestión de pedidos y administración de productos de la Anticuchería Beny. Permite a los clientes realizar pedidos en línea y al administrador gestionar productos, usuarios y órdenes desde un panel administrativo.

## Tecnologías Utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap
* phpMyAdmin

## Funcionalidades

### Cliente

* Registro e inicio de sesión.
* Catálogo de productos.
* Carrito de compras.
* Lista de deseos.
* Realización de pedidos.
* Seguimiento de órdenes.
* Búsqueda de productos.

### Administrador

* Gestión de productos.
* Gestión de pedidos.
* Gestión de usuarios.
* Gestión de mensajes.
* Dashboard administrativo.

## Base de Datos

El proyecto utiliza MySQL para el almacenamiento de información de usuarios, productos y pedidos.

## Instalación

1. Clonar el repositorio.
2. Importar la base de datos en MySQL.
3. Configurar la conexión en los archivos PHP.
4. Ejecutar el proyecto mediante XAMPP o WAMP.
